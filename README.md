## 📖 Overview

This project is an **educational Python-based tool** designed to help understand how a **server behaves under controlled load conditions**.

It is intended for:
- learning basic performance and resilience concepts
- observing timeouts, errors, and recovery behavior
- testing **your own infrastructure** in a safe and responsible way

This repository is **not** created for malicious purposes.

---

## ☁️ Execution Environment

⚠️ **Cloud Shell Only**

This script is designed to be executed **only within Cloud Shell environments** (such as Google Cloud Shell or similar managed shells).

Reasons:
- isolated execution environment  
- temporary sessions  
- reduced risk to local machines  
- easier monitoring and teardown  

Running this tool outside Cloud Shell environments is **not recommended**.
